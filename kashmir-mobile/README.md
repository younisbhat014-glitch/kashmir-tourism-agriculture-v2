# Kashmir Paradise Portal Mobile

React Native mobile application built with Expo Router. It uses the same Railway backend and MongoDB database as the deployed Kashmir Paradise Portal website.

## Run on Expo Go

```powershell
cd kashmir-mobile
npx.cmd expo start
```

Scan the terminal QR code with Expo Go while the phone and laptop are connected to the same network.
